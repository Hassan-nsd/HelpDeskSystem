# Help Desk Management System

A full-stack Help Desk Management System designed to streamline IT support operations by allowing users to create, track, and manage support tickets efficiently. The system provides role-based access control, ticket management, reporting, and file attachment capabilities through a modern web interface.

## 🚀 Features

### Authentication & Authorization

- Secure user authentication
- Role-based access control
- Admin and user permissions management

### Ticket Management

- Create support tickets
- View and update ticket details
- Assign and track ticket status
- Categorize tickets by type
- Manage ticket priorities

### File Attachments

- Upload attachments to tickets
- Download attached files
- Secure file management

### User Management

- Admin-only user management panel
- Create and manage system users
- Assign user roles

### Dashboard & Reporting

- Ticket statistics dashboard
- Category-based reporting
- Visual representation of ticket data
- Performance and activity tracking

### Responsive Design

- Optimized for desktop, tablet, and mobile devices
- Modern and user-friendly interface
